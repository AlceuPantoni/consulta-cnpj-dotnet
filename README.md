# Consulta CNPJ

App para consultar CNPJs utilizando API gratuita (limitada).

## Objetivos

- Fazer triagens iniciais de CNPJs
- Obter dados cadastrais de empresas sem custos

## Requisitos
- Windows 10 ou superior
- Não requer instalação do .NET Runtime (versão publicada como self-contained)
- Conexão com a Internet
